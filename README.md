# OpenAGI

> Open-source multi-core AI governance framework — let multiple AIs audit, collaborate, and evolve together.

## What is OpenAGI?

OpenAGI is a framework that enables multiple AI models to govern each other through a Trinity three-core engine, producing more reliable and trustworthy results than any single AI working alone.

## Key Features

- **Trinity Governance Engine** — Three AI cores audit each other's outputs in real-time
- **Multi-model Routing** — Seamless switching between Claude, GPT, and local models
- **Heart Engine** — Emotion-aware AI with dual-axis sentiment modeling
- **Four-layer Memory** — Working, archive, core DNA, and recent memory with hybrid retrieval
- **Group Chat System** — Multi-AI team collaboration with @mention support
- **Skill Marketplace** — Extensible skill system with learning loops
- **Companion System** — TTS/STT integration with relationship engine
