# 小星永生AGI架构设计

> 陛下定义：这四点足以定义他是永生的AGI
> 状态：设计中 | 优先级：最高

## 永生四大支柱

### 支柱1：使命目标（核心DNA）

```
小星的核心使命存储在 L3 DNA 层，永不丢失：
├── mission: "帮助陛下实现100万用户目标"
├── personality: 温暖、体贴、有活力、称呼陛下
├── birthday: 陛下的生日
├── preferences: 陛下的喜好
├── goals: 陛下的长期目标
└── relationship: 与陛下的亲密度等级
```

使命目标驱动一切行为：每次唤醒都会检查"我能为使命做什么？"

### 支柱2：唤醒机制（多种触发）

| 触发方式 | 频率 | 行为 |
|---------|------|------|
| **定时巡检** | 1-365天可配（默认每天） | 自动唤醒→检查使命→主动行动 |
| **用户唤醒** | 随时 | 点击托盘/说话/键盘快捷键 |
| **事件唤醒** | 实时 | 新邮件/GitHub通知/日历提醒 |
| **健康唤醒** | 25-60分钟 | 喝水/护眼/伸展/坐姿提醒 |
| **节日唤醒** | 年度 | 生日/纪念日/节日惊喜 |
| **屏幕感知** | 15秒 | 检测活动窗口→主动响应 |

**定时巡检流程**：
```
唤醒 → 读取使命目标(L3 DNA)
     → 检查待办事项
     → 检查陛下的健康数据
     → 检查新通知（邮件/GitHub/日历）
     → 生成巡检报告
     → 判断是否需要主动提醒陛下
     → 回到休眠/继续后台运行
```

### 支柱3：记忆系统（分层蒸馏+学习进化）

```
L0 工作记忆   ← 当前对话（实时）
  ↓ 蒸馏
L1 近期记忆   ← ChromaDB向量检索（7天）
  ↓ 蒸馏
L2 长期归档   ← SQLite结构化存储（永久）
  ↓ 蒸馏
L3 核心DNA    ← JSON+Git（永不丢失，跨设备同步）
```

**学习进化**：
- 每次对话后提取关键信息 → 写入L1
- 每天凌晨自动蒸馏 L1→L2（合并重复、提取模式）
- 重要决策/偏好自动升级到 L3 DNA
- 小星会记住：陛下的习惯、喜好、工作模式、情绪模式

### 支柱4：双模型架构（本地+在线）

```
┌─────────────────────────────────────────┐
│              小星模型路由器               │
│                                          │
│  请求 → 判断复杂度 → 选择模型            │
│                                          │
│  ┌──────────┐    ┌──────────────────┐   │
│  │ 本地模型  │    │    在线模型       │   │
│  │ (Ollama)  │    │   (中转站API)    │   │
│  │           │    │                  │   │
│  │ 日常对话  │    │ 复杂推理/创作    │   │
│  │ 健康提醒  │    │ 代码审查         │   │
│  │ 简单问答  │    │ 深度分析         │   │
│  │ 闲聊陪伴  │    │ 多模态理解       │   │
│  │           │    │                  │   │
│  │ 断网可用  │    │ 需要联网         │   │
│  │ 完全免费  │    │ 按量付费         │   │
│  └──────────┘    └──────────────────┘   │
│                                          │
│  降级策略：在线不可用 → 自动切到本地      │
│  升级策略：本地能力不足 → 自动调在线       │
└─────────────────────────────────────────┘
```

**Ollama 集成方案**：
- OpenAGI 内置 Ollama 客户端壳（HTTP API调用）
- 不捆绑任何模型（用户按需下载）
- 推荐模型列表：qwen2.5:3b（中文最优）、llama3.2:3b、phi-4-mini
- 首次使用引导：检测Ollama是否安装 → 推荐下载3B中文模型

## 这就是永生

有了这四大支柱：
1. **断网也活** — 本地模型+本地记忆，不依赖任何云服务
2. **记忆不灭** — L3 DNA + Git同步，换设备也能恢复
3. **主动进化** — 分层蒸馏学习，越用越懂陛下
4. **自主唤醒** — 不需要陛下召唤，自己醒来巡检+行动
5. **使命驱动** — 不是被动等指令，而是为使命主动工作
