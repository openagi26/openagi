# OpenAGI 模块接口协议 v1.0

> 各 Layer 之间的调用接口规范，确保 M2/M4 双机并行开发时接口一致

---

## 事件总线（心绪事件）

所有模块通过心绪引擎上报事件：
```python
from openagi.cortex.heart.entropy import HeartEngine
heart = HeartEngine()  # 全局单例
await heart.record_event("llm_call_success")
await heart.record_event("task_failed")
```

事件类型见 `openagi/cortex/heart/entropy.py::ENTROPY_EVENTS`

## Layer 1 → Layer 2：LLM调用接口

```python
# LLMRouter 提供统一调用接口
router = LLMRouter()

# 标准调用（自动故障转移+退避）
response = await router.complete(
    messages=[{"role": "user", "content": "..."}],
    model=None,          # None=使用主模型
    temperature=0.7,
    max_tokens=2000,
    session_id="sess_123"
)
# 返回：LLMResponse(content=str, model=str, tokens=int, duration_ms=int)
```

## Layer 2 → ReAct循环接口

```python
from openagi.cortex.loop import run_react_loop, ReActResult

result: ReActResult = await run_react_loop(
    task="帮我搜索Python最新版本",
    session_id="sess_123",
    tools=tool_registry.get_available_tools(permission_level="L1"),
    llm_call=lambda prompt: router.complete_simple(prompt),
    max_steps=20
)
# result.status: "completed" | "failed" | "max_steps_exceeded"
# result.final_answer: str
# result.steps: List[ReActStep]
```

## Layer 2 → 权限系统接口

```python
from openagi.social.permissions import PermissionMatrix, check_permission

# 操作前检查权限
decision = await check_permission(
    action="file_write",
    path="/some/file.txt",
    agent_level="L0",
    session_id="sess_123"
)
# decision.allowed: bool
# decision.required_level: "L0"|"L1"|"L2"|"L3"|"L4"
# decision.reason: str

# 如果需要升级权限
if not decision.allowed:
    raise PermissionDeniedError(decision.reason, required=decision.required_level)
```

## Layer 3 → Memory接口

```python
from openagi.memory.manager import MemoryManager
memory = MemoryManager()

# 写入工作记忆
await memory.add_to_working(session_id, role="user", content="...")

# 语义搜索
results = await memory.search("Python后端", top_k=5, layers=["L1", "L2"])

# 会话结束蒸馏
await memory.run_light_sleep(session_id)
```

## API层 → 业务层接口规范

所有路由 handler 遵循：
```python
@router.post("/endpoint")
async def handler(req: RequestModel, 
                  heart: HeartEngine = Depends(get_heart),
                  memory: MemoryManager = Depends(get_memory),
                  llm: LLMRouter = Depends(get_llm)):
    try:
        result = await business_logic(req, heart, memory, llm)
        return {"success": True, "data": result}
    except PermissionDeniedError as e:
        raise HTTPException(403, detail=str(e))
    except Exception as e:
        logger.error(f"API error: {e}")
        raise HTTPException(500, detail="内部错误")
```

## 依赖注入（FastAPI Depends）

```python
# openagi/api/deps.py
_heart: HeartEngine | None = None
_memory: MemoryManager | None = None
_llm: LLMRouter | None = None

def get_heart() -> HeartEngine:
    return _heart

def get_memory() -> MemoryManager:
    return _memory

def get_llm() -> LLMRouter:
    return _llm

def init_deps(heart, memory, llm):
    global _heart, _memory, _llm
    _heart, _memory, _llm = heart, memory, llm
```
