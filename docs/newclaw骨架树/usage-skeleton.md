# Usage 页面 — 骨架树（去叶留枝版）

```
/usage
│
├── 页面头部
│   ├── 标题："使用情况"
│   ├── 副标题："Token 用量、API 调用和费用统计"
│   └── [刷新] 按钮
│       └── （加载中时：刷新图标旋转动画 + 按钮禁用）
│
├── 统计总览区（四格卡片，2列/4列响应式）
│   │
│   ├── 卡片①：总 Token
│   │   ├── 标题："总 Token"
│   │   ├── 数值：格式化显示（K / M 缩写）
│   │   └── 明细
│   │       ├── 输入 N（↑图标）
│   │       └── 输出 N（↓图标）
│   │
│   ├── 卡片②：API 调用
│   │   ├── 标题："API 调用"
│   │   ├── 数值：累计请求次数（整数）
│   │   └── 副文字："累计请求次数"
│   │
│   ├── 卡片③：预估费用
│   │   ├── 标题："预估费用"
│   │   ├── 数值："$0.00"（保留两位小数）
│   │   └── 副文字："基于 Token 估算"
│   │
│   └── 卡片④：模型数
│       ├── 标题："模型数"
│       ├── 数值：已使用的模型种数（整数）
│       └── 副文字："已使用的模型"
│
└── 模型用量明细区
    ├── 区域标题："模型用量明细"
    │
    ├── 有数据时：模型行列表（按总 Token 降序排列）
    │   └── 每行
    │       ├── 模型名称（文本）
    │       ├── 提供商徽章（Badge）
    │       ├── 总 Token 数（格式化 K/M）
    │       ├── 用量占比进度条（百分比宽度，最小 1%）
    │       ├── 调用次数："N 次调用"
    │       └── 百分比："X.X%"
    │
    └── 空状态（无数据时）
        ├── 警告图标
        ├── 主文字："暂无使用数据"
        └── 副文字："开始使用 AI 对话后，这里会显示用量统计"
```

## 数据流骨架

```
用量数据
  → 页面挂载时立即拉取 /api/gateway/usage
  → 定时刷新：每 30 秒自动拉取一次
  → 手动刷新：点击 [刷新] 按钮
  → 拉取失败时降级为零值（totalInputTokens=0, totalOutputTokens=0, totalRequests=0, totalCostUsd=0, modelBreakdown=[]）
  → 页面卸载时清除定时器
```

## 状态枚举

```
加载状态
  ├── loading=true（首次加载 / 点击刷新）：刷新图标旋转
  └── loading=false：正常显示数据

数据状态
  ├── 有模型数据（modelBreakdown.length > 0）：显示明细列表
  └── 无模型数据（modelBreakdown.length === 0）：显示空状态
```

## 说明

- 页面无 i18n 依赖，所有文本硬编码中文
- 页面无全局 store 依赖，直接通过 `hostApiFetch` 调用网关 API
- 无弹窗、无 Toast、无表单交互
- 数值格式化规则：≥1,000,000 显示为 `N.NM`；≥1,000 显示为 `N.NK`；其余直接显示整数
