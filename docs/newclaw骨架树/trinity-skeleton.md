## 数据流骨架

```
Trinity Store（Zustand）
  ├── dashboard（仪表盘数据）→ 页面加载时拉取
  ├── genesisStatus（创世状态）→ 页面加载时检测
  └── isLoading（加载状态）→ 控制按钮禁用状态

页面级本地状态（各子页面独立管理）
  ├── GoalPage：currentGoal / goalHistory / runnerState / showForm / confirmAbandon
  ├── GovernancePage：tab 切换 / evidence / debts / playbooks / pooTasks / confirmClearing
  ├── MarketPage：tab 切换 / predictions / prophetStats / listings / marketStats / confirmPurchase
  ├── SwarmPage：peers / stats / connectForm / reportForm / showReportForm
  ├── BlockchainPage：tab 切换 / chainState / blocks / rates / priceHistory / dividendSummary / confirmPayout
  └── SettingsPage：各配置字段本地状态 / saving / saved

游戏模式（store 层）
  ├── debate（辩论模式）
  ├── competition（竞争模式）
  └── refinement（精炼模式）

身份管理（store 层）
  ├── unlockIdentity(passphrase) → 解锁
  └── lockIdentity() → 锁定
```
