## 开发者模式说明

- 入口：设置页面 → 通用 → 开发者模式开关
- 影响本页面的功能：
  - Model Studio 提供商在选择网格中可见
  - 部分提供商额外显示模型 ID 字段
  - Token 用量条目显示 [查看内容] 按钮
  - 点击条目可打开用量详情弹窗

## 数据流骨架

```
Token 用量数据
  → 网关运行中时自动拉取
  → 刷新触发：每15秒定时 / 窗口获得焦点 / 页面从后台切回前台
  → 重试：最多2-3次，间隔1.5秒
  → 过滤隐藏来源 → 按时间窗口筛选 → 按维度分组 → 分页显示

提供商操作
  → 添加：选类型 → 填表单 → 验证 API Key → 创建账户 → 刷新列表
  → 编辑：展开表单 → 修改字段 → 验证 → 更新账户 → 刷新列表
  → 删除：确认对话框 → 删除账户 → 刷新列表
  → 设默认：直接设置 → 刷新列表
  → OAuth：发起授权 → 等待设备码 → 浏览器确认 → 回调成功 → 刷新列表
        ↳ 可取消 / 可手动输入码 / 失败可重试
```
