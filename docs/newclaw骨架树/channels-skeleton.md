# Channels 页面 — 骨架树（去叶留枝版）

```
/channels
│
├── 页面头部
│   ├── 标题："消息频道"
│   ├── 副标题："统一管理消息频道、账号、账号与智能体的绑定关系，以及频道默认账号"
│   └── [刷新] 按钮（网关未运行时禁用，刷新中显示旋转动画）
│
├── 全局警告横幅（网关未运行时显示）
│   └── 警告："网关未运行。未启用网关时无法管理频道。"
│
├── 错误横幅（加载失败时显示）
│   └── 错误消息文本
│
├── 已配置频道区（有已配置频道时显示）
│   ├── 区域标题："已配置频道"
│   │
│   └── 频道组卡片列表（每种频道类型一张卡片）
│       └── 频道组卡片
│           ├── 卡片头部
│           │   ├── 频道图标（Telegram / Discord / WhatsApp / WeChat / DingTalk / Feishu / WeCom / QQ Bot 专属图标，其他用 emoji）
│           │   ├── 频道名称（显示名，如 "Telegram"）
│           │   ├── 频道类型 ID（如 "telegram"，灰色小字）
│           │   ├── 连接状态指示点
│           │   │   ├── 绿色："connected（已连接）"
│           │   │   ├── 黄色闪烁："connecting（连接中）"
│           │   │   ├── 红色："error（异常）"
│           │   │   └── 灰色："disconnected（未连接）"
│           │   ├── [添加账号] 按钮 → 打开配置弹窗（新建账号模式）
│           │   └── [删除频道] 图标按钮 → 弹出确认对话框（删除整个频道组）
│           │
│           └── 账号列表（该频道下所有账号，每行一个）
│               └── 账号行
│                   ├── 账号显示名
│                   │   └── 特殊情况：accountId 为 "default" 且 name 与 accountId 相同时，显示 "主账号"
│                   ├── 错误信息（有 lastError 时显示，红色）
│                   ├── "绑定 Agent（智能体）" 标签
│                   ├── Agent 绑定下拉选择器
│                   │   ├── 选项："未绑定"（空值）
│                   │   └── 选项列表：所有可用 Agent 名称（onChange → 立即调用绑定/解绑 API → Toast）
│                   ├── [编辑] 按钮 → 预拉取账号配置 → 打开配置弹窗（编辑模式）
│                   └── [删除账号] 图标按钮 → 弹出确认对话框（仅删除该账号）
│
├── 支持的频道区（始终显示，仅列出尚未配置的频道类型）
│   ├── 区域标题："支持的频道"
│   │
│   └── 频道卡片网格（2列，每个未配置的主频道类型一张卡片）
│       └── 频道卡片按钮（点击 → 打开配置弹窗）
│           ├── 频道图标
│           ├── 频道名称
│           ├── [插件] 徽章（isPlugin 为 true 时显示）
│           └── 频道描述文字（来自 i18n meta 描述）
│
│   ※ 主频道类型固定顺序：
│     telegram / discord / whatsapp / wechat（插件）/ dingtalk（插件）/
│     feishu（插件）/ wecom（插件）/ qqbot
│
├── 频道配置弹窗（点击频道卡片或"添加账号"/"编辑"后弹出）
│   ├── 弹窗头部
│   │   ├── 标题（三种模式）
│   │   │   ├── 新增频道（未选类型）："添加频道"
│   │   │   ├── 新增/配置："配置 {{频道名}}"
│   │   │   └── 编辑已有配置："更新 {{频道名}}"
│   │   ├── 副标题描述（三种情况）
│   │   │   ├── 已有配置："更新您现有的配置"
│   │   │   ├── 未选类型："选择要配置的频道类型"
│   │   │   └── 已选类型：该频道的功能描述文字
│   │   └── [关闭 ×] 按钮 / 点击遮罩层关闭
│   │
│   ├── 第一步：频道类型选择网格（未预先指定类型时显示）
│   │   └── 每个主频道类型卡片
│   │       ├── 频道图标
│   │       ├── 频道名称
│   │       ├── [插件] 徽章（isPlugin 时）
│   │       ├── 频道描述
│   │       ├── 连接类型标签："二维码" / "令牌（Token）"
│   │       └── [已配置] 绿色徽章（该类型已有配置时）
│   │
│   ├── 加载状态（预拉取已有配置中）
│   │   └── 加载动画 + 文字："正在加载配置..."
│   │
│   └── 第二步：配置表单（已选定频道类型后显示）
│       │
│       ├── 已有配置提示横幅（isExistingConfig 时）
│       │   └── "您已配置过此频道"
│       │
│       ├── 如何连接说明区
│       │   ├── 标题："如何连接"
│       │   ├── 频道描述文字
│       │   ├── [查看文档] 按钮 → 打开外部文档链接
│       │   └── 步骤列表（有序编号，各频道各自的操作步骤）
│       │
│       ├── 账号 ID 输入框（allowEditAccountId 为 true 时显示）
│       │   ├── 标签："账号 ID"
│       │   ├── 输入框（占位符："例如：feishu-sales-bot"）
│       │   └── 提示文字："可自定义账号 ID，用于区分同一频道下的多个账号。"
│       │
│       ├── 配置字段区（按各频道 configFields 定义动态渲染）
│       │   └── 每个字段
│       │       ├── 字段标签（必填字段带红色 * 标记）
│       │       ├── 文本输入框 或 密码输入框（type 决定）
│       │       │   └── 密码类型：右侧 [显示/隐藏] 切换按钮（眼睛图标）
│       │       ├── 字段描述文字（有时）
│       │       └── 环境变量提示："环境变量: {{VAR_NAME}}"（有 envVar 时）
│       │
│       │   ※ 各频道字段一览：
│       │   ┌─────────────────┬─────────────────────────────────────────────────────┐
│       │   │ 频道             │ 必填字段 → 可选字段                                  │
│       │   ├─────────────────┼─────────────────────────────────────────────────────┤
│       │   │ Telegram        │ 机器人令牌（密码）* + 允许用户 ID（文本）*             │
│       │   │ Discord         │ 机器人令牌（密码）* + 服务器 ID（文本）* → 频道 ID     │
│       │   │ WhatsApp        │ 无字段（纯二维码流程）                                 │
│       │   │ WeChat          │ 无字段（纯二维码流程，插件）                           │
│       │   │ DingTalk        │ Client ID* + Client Secret（密码）* → RobotCode/CorpId/AgentId │
│       │   │ Feishu/Lark     │ App ID（文本）* + App Secret（密码）*                 │
│       │   │ WeCom           │ Bot ID（文本）* + Secret（密码）*                     │
│       │   │ QQ Bot          │ App ID（文本）* + Client Secret（密码）*               │
│       │   │ Signal          │ 手机号码（文本）*                                      │
│       │   │ iMessage        │ 服务器地址（文本）* + 密码（密码）*                    │
│       │   │ Matrix          │ Homeserver 地址（文本）* + 访问令牌（密码）*            │
│       │   │ LINE            │ 频道访问令牌（密码）* + 频道密钥（密码）*               │
│       │   │ Microsoft Teams │ 应用 ID（文本）* + 应用密码（密码）*                   │
│       │   │ Google Chat     │ 服务账号 JSON 路径（文本）*                             │
│       │   │ Mattermost      │ 服务器地址（文本）* + 机器人访问令牌（密码）*           │
│       │   └─────────────────┴─────────────────────────────────────────────────────┘
│       │
│       ├── 验证结果面板（点击"验证配置"后显示）
│       │   ├── 成功状态
│       │   │   ├── 标题："凭证已验证"
│       │   │   └── 详情信息（如 Bot 用户名、服务器名、频道名）
│       │   └── 失败状态
│       │       ├── 标题："验证失败"
│       │       ├── 错误列表
│       │       └── 警告列表（有时）
│       │
│       ├── 二维码显示区（connectionType 为 "qr"，点击生成后显示）
│       │   ├── 二维码图片（base64 或 URL）
│       │   ├── 提示文字："使用 {{频道名}} 扫描此二维码"
│       │   └── [刷新代码] 按钮 → 重新生成二维码
│       │
│       └── 操作按钮区
│           ├── [验证配置] 按钮（仅 token 类型 且 非 feishu 时显示）
│           │   ├── 验证中：加载动画 + "正在验证..."
│           │   └── 默认：盾牌图标 + "验证配置"
│           └── 主操作按钮（根据状态变化）
│               ├── 二维码类型（未生成）："生成二维码"
│               ├── 二维码类型（生成中）：加载动画 + "正在生成二维码..."
│               ├── Token 类型（新增）：对勾图标 + "保存并连接"
│               ├── Token 类型（更新）：对勾图标 + "更新并重新连接"
│               └── 保存/连接中：加载动画 + "正在验证并保存..."
│               ※ 表单未填满必填字段时禁用；账号 ID 为空时禁用
│
├── 删除确认对话框（点击任意删除按钮后弹出）
│   ├── 标题："Confirm（确认）"
│   ├── 消息
│   │   ├── 删除账号："确定要删除该账号吗？"
│   │   └── 删除频道："确定要删除此频道吗？"
│   ├── [删除] 按钮（危险样式）→ 执行删除 → Toast 反馈
│   └── [取消] 按钮 → 关闭对话框
│
├── 全页加载状态（初次加载 且 无缓存数据时）
│   └── 居中大号加载旋转动画
│
└── Toast 消息（操作反馈，共11种）
    ├── "{{name}} 连接成功"（二维码连接成功）
    ├── "{{name}} 连接失败: {{error}}"（二维码失败）
    ├── "频道 {{name}} 已保存"
    ├── "正在连接 {{name}}..."
    ├── "配置已保存，但刷新页面数据失败，请手动刷新查看最新状态"
    ├── "配置失败: {{error}}"
    ├── "账号绑定已更新"
    ├── "账号已删除"
    ├── "频道已删除"
    ├── "账号 ID 不能为空"
    └── "账号 ID {{accountId}} 已存在"
```

## 频道连接类型说明

```
连接类型    频道
──────────  ──────────────────────────────────────────────
token       Telegram / Discord / DingTalk / Signal /
            Feishu / WeCom / iMessage / Matrix / LINE /
            Microsoft Teams / Mattermost / QQ Bot
qr          WhatsApp / WeChat
webhook     Google Chat
```

## 插件频道标识

以下频道带有 [插件] 徽章，需要通过 OpenAGI 插件系统接入：
- WeChat（微信）
- DingTalk（钉钉）
- Feishu / Lark（飞书）
- WeCom（企业微信）
- Matrix
- LINE
- Microsoft Teams
- Mattermost

## 数据流骨架

```
页面数据加载
  → 并行拉取：/api/channels/accounts（频道账号列表）+ /api/agents（Agent 列表）
  → 触发时机：页面初始化 / 手动刷新 / 收到 gateway:channel-status 事件 / 网关从非运行切换到运行
  → 加载中且有缓存数据时：显示旧数据并在刷新按钮上显示旋转动画
  → 加载中且无缓存数据：全页居中加载动画

账号绑定操作
  → 选择 Agent → PUT /api/channels/binding（绑定）或 DELETE /api/channels/binding（解绑）
  → Toast "账号绑定已更新" / "配置失败: {{error}}"
  → 成功后刷新页面数据

频道删除操作
  → 确认 → DELETE /api/channels/config/{{channelType}}（删除频道）
             或 DELETE /api/channels/config/{{channelType}}?accountId=...（删除账号）
  → 乐观更新（立即从列表移除）→ 1200ms 后再次拉取（等待主进程防抖完成）
  → Toast "频道已删除" / "账号已删除" / "配置失败: {{error}}"

频道配置保存（token 类型）
  → 验证凭证 → POST /api/channels/config（保存配置）
  → 成功 → 关闭弹窗 → 刷新页面数据
  → Toast "频道 {{name}} 已保存" + "正在连接 {{name}}..."

频道配置保存（qr 类型）
  → POST /api/channels/{{type}}/start（发起二维码会话）
  → 监听 host 事件：{{type}}:qr → 显示二维码图片
  → 监听 host 事件：{{type}}:success → 保存配置 → 关闭弹窗
  → 监听 host 事件：{{type}}:error → Toast 错误
  → 弹窗关闭时：取消会话 POST /api/channels/{{type}}/cancel

验证功能（非 feishu 的 token 类型）
  → POST /api/channels/credentials/validate（验证凭证）
  → 验证通过：绿色面板 + Bot/服务器/频道详情
  → 验证失败：红色面板 + 错误列表 + 警告列表
```
