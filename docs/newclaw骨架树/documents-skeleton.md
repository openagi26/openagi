# Documents 页面 — 骨架树（去叶留枝版）

```
/documents
│
├── 页面头部
│   ├── 标题："文档"
│   ├── 副标题："知识库文档与配置文件 · N 个文件"
│   └── [刷新] 按钮（加载中显示旋转动画）
│
├── 搜索与过滤区
│   ├── 搜索输入框
│   │   └── placeholder："搜索文档名称或描述..."
│   └── 类型过滤器（标签切换组）
│       ├── [全部]（始终显示）
│       ├── [技能]（有数量时显示计数：技能 (N)）
│       ├── [智能体]（有数量时显示计数：智能体 (N)）
│       └── [配置]（有数量时显示计数：配置 (N)）
│
├── 文档列表区
│   ├── 有匹配文档时：文档卡片列表
│   │   └── 每张卡片
│   │       ├── 类型图标（按文件类型变化）
│   │       │   ├── skill（技能文件）→ 代码图标
│   │       │   ├── agent（智能体）→ 书本图标
│   │       │   ├── config（配置文件）→ 文件图标
│   │       │   └── other（其他）→ 文本文件图标
│   │       ├── 文件名称（截断处理）
│   │       ├── 类型徽章
│   │       │   ├── "技能文件"（skill 类型）
│   │       │   ├── "智能体"（agent 类型）
│   │       │   ├── "配置文件"（config 类型）
│   │       │   └── "其他"（other 类型）
│   │       ├── 元数据行
│   │       │   ├── 文件路径（截断处理）
│   │       │   ├── 文件大小（自动换算：B / KB / MB）
│   │       │   └── 最后修改时间（中文日期格式）
│   │       └── （可选）文件描述（有 description 字段时显示）
│   │
│   └── 空状态（无匹配文档时）
│       ├── 警告图标
│       ├── （搜索词或过滤器激活时）："没有匹配的文档"
│       ├── （无任何文档时）："暂无文档"
│       └── 提示文字："文档会在配置智能体和技能后自动出现"
│
└── 数据状态
    ├── 加载中：[刷新] 按钮图标旋转，列表为空
    └── 加载完成：显示文档列表或空状态
```

## 文件类型说明

```
类型值      显示标签    图标
skill      技能文件    代码图标（Code）
agent      智能体      书本图标（BookOpen）
config     配置文件    文件图标（File）
other      其他        文本图标（FileText）
```

## 过滤逻辑

```
类型过滤
  → "all"：显示全部文档
  → "skill" / "agent" / "config"：仅显示对应 type 的文档

搜索过滤（与类型过滤叠加）
  → 匹配字段：文件名称（name）、文件描述（description）
  → 大小写不敏感

计数徽章
  → 过滤按钮（非"全部"）在有对应类型文档时显示数量
```

## 数据流骨架

```
文档数据
  → 页面挂载时自动请求：GET /api/gateway/documents
  → 点击 [刷新] 按钮手动重新请求
  → 请求失败：静默处理，显示空列表
  → 请求成功：解析 documents 数组 → 前端过滤 → 渲染列表

无外部 Store 依赖（直接通过 hostApiFetch 调用网关 API）
无 Toast 消息
无弹窗
无分页（列表全量渲染）
```
