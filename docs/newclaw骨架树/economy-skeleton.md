## 经济数据结构骨架

```
TrinityEconomy（钱包状态）
  ├── balance：当前 New.B 余额
  ├── totalEarned：历史总收入
  ├── totalSpent：历史总支出
  ├── totalStaked：已质押金额
  ├── halvingEpoch：当前减半纪元
  ├── currentRewardRate：当前每任务奖励率
  └── recentTransactions[]：最近交易记录
      ├── id / timestamp / type / amount / balance / description
```
