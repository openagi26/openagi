# Instances 页面 — 骨架树（去叶留枝版）

```
/instances
│
├── 页面头部
│   ├── 标题："实例管理"
│   ├── 副标题："查看和管理运行中的服务实例"
│   └── [刷新] 按钮
│       └── （加载中时图标旋转 + 按钮禁用）
│
├── 实例列表区
│   │
│   ├── 有实例时：实例卡片列表
│   │   └── 每张卡片
│   │       ├── 服务器图标
│   │       ├── 实例名称（如"OpenAGI 网关"、"Claude Code 引擎"）
│   │       ├── 状态徽章
│   │       │   ├── "运行中"（绿点）
│   │       │   ├── "已停止"（灰点）
│   │       │   └── "错误"（红点）
│   │       └── 元数据信息行（按字段是否存在条件显示）
│   │           ├── （有端口时）端口号："端口 XXXXX"
│   │           ├── （有PID时）进程ID："PID XXXXX"
│   │           ├── （有内存时）内存占用："XXX MB"
│   │           └── （有启动时间时）启动时间：本地化时间字符串
│   │
│   └── 无实例时：空状态
│       ├── 警告图标
│       └── 提示文字："暂无运行中的实例"
│
└── 数据刷新机制（无UI，后台运行）
    ├── 页面挂载时：立即拉取一次
    └── 定时轮询：每 10 秒自动刷新
```

## 实例数据字段说明

```
字段名         类型            显示条件
id            string          内部标识，不展示
name          string          始终显示
type          string          内部类型，不展示
status        running/stopped/error   始终显示（徽章）
port          number?         有值时显示
pid           number?         有值时显示
memoryMb      number?         有值时显示（单位 MB）
startedAt     string?         有值时显示（格式化为本地时间）
uptime        number?         字段存在但当前未渲染
cpuPercent    number?         字段存在但当前未渲染
```

## 默认兜底实例（API 无响应时）

```
实例1
  id:     gateway
  名称:   OpenAGI 网关
  类型:   gateway
  状态:   运行中
  端口:   18799
  启动时间: 当前时间

实例2
  id:     code-engine
  名称:   Claude Code 引擎
  类型:   code-engine
  状态:   运行中
```

## 数据流骨架

```
页面挂载
  → 调用 /api/gateway/instances
  → 成功：渲染返回的实例列表
  → 失败（API 异常）：渲染默认兜底实例（网关 + Code 引擎）
  → 每 10 秒重复上述流程
  → 页面卸载时清除定时器
```

## 当前未实现功能（字段预留但无 UI）

```
- CPU 占用率（cpuPercent）：字段已定义，未渲染
- 运行时长（uptime）：字段已定义，未渲染
- 实例操作（启动/停止/重启）：无操作按钮
- 实例详情：无点击展开/跳转
- 日志查看：无入口
- 错误详情：error 状态仅显示徽章，无详情说明
```
