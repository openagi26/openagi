# Skills 页面 — 骨架树（去叶留枝版）

```
/skills
│
├── 页面头部
│   ├── 标题："技能"
│   ├── 副标题："浏览和管理 AI 能力"
│   └── [打开技能文件夹]（仅在有已安装技能时显示）
│
├── 网关警告横幅（网关未运行超过 1.5 秒后显示）
│   └── "网关未运行。没有活跃的网关，无法加载技能。"
│
├── 控制栏（搜索 + 筛选 + 批量操作 + 动作按钮）
│   │
│   ├── 搜索框
│   │   ├── 占位符："搜索技能..."
│   │   ├── 搜索范围：名称 / 描述 / ID / slug / 作者
│   │   └── [×] 清空搜索（有输入时显示）
│   │
│   ├── 来源筛选标签组（互斥单选）
│   │   ├── [全部 (N)]
│   │   ├── [内置 (N)]
│   │   └── [市场 (N)]
│   │
│   └── 右侧动作按钮组
│       ├── [批量启用可见项]
│       ├── [批量禁用可见项]
│       ├── [安装技能] → 打开安装面板（Sheet）
│       └── [刷新图标]（网关未运行时禁用）
│
├── 错误横幅（获取技能失败时显示）
│   ├── fetchTimeoutError："获取技能列表超时，请检查网络。"
│   ├── fetchRateLimitError："获取技能列表请求过于频繁，请稍后再试。"
│   └── 其他错误：直接显示错误文本
│
├── 技能列表区（可滚动）
│   │
│   ├── 加载状态（初始加载中）
│   │   └── 全屏居中加载动画（LoadingSpinner）
│   │
│   ├── 空状态（filteredSkills.length === 0）
│   │   ├── （有搜索词时）"尝试不同的搜索词"

│   │
│   └── 技能条目列表（排序：已启用 > 未启用；核心 > 非核心；字母序）
│       └── 每条技能行（点击 → 打开详情面板）
│           ├── 技能图标（emoji，默认 🧩）
│           ├── 技能名称
│           ├── 类型标识
│           │   ├── 🔒 锁图标（核心系统技能）

│           ├── slug 徽章（当 slug ≠ 名称时显示）
│           ├── 描述文字（单行截断）
│           ├── 底部元信息行
│           │   ├── 来源徽章（见来源标签表）
│           │   └── baseDir 路径（路径不可用时："路径不可用"）
│           ├── 版本号 vX.X.X（有版本时显示）
│           └── 启用/禁用开关（核心技能禁用，不可操作）
│
├── 安装技能面板（Sheet，从右侧滑入）
│   ├── 头部
│   │   ├── 标题："安装技能"
│   │   ├── 副标题："默认展示 Explore，输入关键词后执行搜索。"
│   │   ├── 搜索框：占位符"搜索市场..."  + [×] 清空
│   │   └── 来源标签（只读禁用）："来源: ClawHub"
│   │
│   ├── 搜索中状态
│   │   └── 居中加载动画 + "正在搜索 ClawHub..."
│   │
│   ├── 搜索错误横幅（searchError 时显示）
│   │   ├── searchTimeoutError："搜索超时，请检查网络。您也可访问 ClawHub.ai 搜索并下载压缩包，解压到 {path}"
│   │   ├── searchRateLimitError："搜索请求过于频繁。您也可访问 ClawHub.ai 搜索并下载压缩包，解压到 {path}"
│   │   └── 其他错误："ClawHub 搜索失败。请检查您的连接或安装。"
│   │
│   ├── 搜索结果列表（有结果时）
│   │   └── 每条市场技能
│   │       ├── 图标（📦）
│   │       ├── 技能名称
│   │       ├── 作者（有时显示）
│   │       ├── 描述（单行截断）
│   │       ├── 版本号（有时显示）
│   │       ├── （点击条目行）→ 跳转到 ClawHub.ai/s/{slug} 外链
│   │       ├── （已安装）[🗑️ 卸载图标]（卸载中显示 LoadingSpinner）
│   │       └── （未安装）[安装]（安装中显示 LoadingSpinner）
│   │
│   └── 空提示区（无结果且无错误时）
│       ├── （有搜索词且无结果）"未找到匹配的技能。"
│       └── （无搜索词）"搜索新技能以扩展您的能力。"
│
└── 技能详情面板（Sheet，从右侧滑入，点击技能行触发）

    ├── 头部信息区
    │   ├── 技能图标（大号 emoji，默认 🔧）
    │   ├── 核心技能标识（🔒 小图标叠加，isCore 时显示）

    │   ├── 版本徽章：vX.X.X
    │   ├── 类型徽章："核心系统" / "内置" / "用户安装"
    │   └── 描述文字（有时显示）
    │
    ├── 来源区（所有技能显示）
    │   ├── 区域标题："来源"
    │   ├── 来源徽章（见来源标签表）
    │   ├── baseDir 路径输入框（只读，路径不可用时："路径不可用"）
    │   ├── [复制图标] → 复制 baseDir 到剪贴板（baseDir 不存在时禁用）
    │   └── [文件夹图标] → 打开实际目录（baseDir 不存在时禁用）
    │
    ├── API 密钥区（非核心技能显示）
    │   ├── 区域标题："API 密钥"（带钥匙图标）
    │   ├── 密码输入框：占位符"输入 API 密钥（可选）"

    │

    │   │
```

## 数据流骨架

```
技能列表数据
  → 网关运行时自动拉取（fetchSkills）
  → 数据来源合并：
      1. Gateway RPC skills.status → 运行中技能状态
      2. Host API /api/clawhub/list → 磁盘已安装技能
      3. Host API /api/skills/configs → 本地技能配置（apiKey + env）
  → 三路合并去重 → 展示列表

筛选/排序规则
  → 搜索词过滤（名称/描述/ID/slug/作者）
  → 来源类型过滤（全部/内置/市场）
  → 排序：已启用优先 → 核心优先 → 字母序

技能操作
  → 启用/禁用：Gateway RPC skills.update → 本地 store 更新
  → 批量启用/禁用：遍历可见技能逐个操作，汇总成功数报告
  → 安装：Host API /api/clawhub/install → 自动 enableSkill → fetchSkills
  → 卸载：Host API /api/clawhub/uninstall → fetchSkills
  → 保存配置：IPC skill:updateConfig（apiKey + env）→ fetchSkills
  → 打开文件夹：IPC shell:openPath（技能总目录）
  → 打开实际目录：Host API /api/clawhub/open-path（单技能目录）
  → 打开手册：Host API /api/clawhub/open-readme → 编辑器
  → 复制路径：navigator.clipboard.writeText(skill.baseDir)
  → 跳转 ClawHub：IPC shell:openExternal(clawhub.ai/s/{slug})

安装面板搜索
  → 输入防抖 300ms → Host API /api/clawhub/search
  → 面板关闭时停止搜索
  → 打开时自动触发空查询（加载推荐列表）
```

## 条件可见规则汇总

```
组件/功能                         显示条件
--------------------------------  -------------------------------------
[打开技能文件夹] 按钮             有至少 1 个非内置技能（hasInstalledSkills）
网关警告横幅                      网关 state ≠ running，且超过 1.5 秒
错误横幅（列表区）                error 状态非空
类型锁图标（🔒）                  skill.isCore = true
类型拼图图标（🧩）                skill.isBundled = true && !skill.isCore
slug 徽章                         skill.slug 存在且 ≠ skill.name
来源区                            所有技能
API 密钥区                        !skill.isCore
环境变量区                        !skill.isCore
外部链接区                        skill.slug 存在 && !skill.isBundled && !skill.isCore
[保存配置] 按钮                   !skill.isCore
[卸载] 按钮                       !skill.isCore && !skill.isBundled && slug 存在
[启用]/[禁用] 按钮                !skill.isCore && (skill.isBundled 或无 slug)
开关禁用状态                      skill.isCore = true
[刷新] 按钮禁用状态               网关未运行
[复制路径] 禁用状态               !skill.baseDir
[打开目录] 禁用状态               !skill.baseDir
```
