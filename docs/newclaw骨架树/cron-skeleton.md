# Cron 页面 — 骨架树（去叶留枝版）

```
/cron
│
├── 页面头部
│   ├── 标题："定时任务"
│   ├── 副标题："通过定时任务自动化 AI 工作流"
│   ├── [刷新] 按钮（网关未运行时禁用）
│   └── [新建任务] 按钮（网关未运行时禁用）
│
├── 告警区（条件）
│   ├── 网关未运行警告（isGatewayRunning=false 时）
│   │   └── "网关未运行。没有活跃的网关，无法管理定时任务。"
│   └── 加载错误提示（error 非空时）
│       └── 显示错误文本
│
├── 统计卡片栏（4 列）
│   ├── 任务总数 — 数字 + "任务总数"
│   ├── 运行中 — 数字 + "运行中"（统计 enabled=true 的任务）
│   ├── 已暂停 — 数字 + "已暂停"（统计 enabled=false 的任务）
│   └── 失败 — 数字 + "失败"（统计 lastRun.success=false 的任务）
│
├── 任务列表区
│   ├── 加载中状态（loading=true）
│   │   └── 全屏居中加载动画（LoadingSpinner，跳出列表渲染）
│   │
│   ├── 空状态（safeJobs.length=0）
│   │   ├── 标题："暂无定时任务"
│   │   ├── 描述："创建定时任务以自动化 AI 工作流。任务可以在指定时间发送消息、运行查询或执行操作。"
│   │   └── [创建第一个任务]（网关未运行时禁用）→ 打开创建弹窗
│   │
│   └── 任务卡片列表（两列网格，每个任务一张卡）
│       └── 任务卡片（CronJobCard）
│           ├── 点击卡片 → 打开编辑弹窗
│           │
│           ├── 头部区域
│           │   ├── 时钟图标（enabled=true 时前景色 / false 时灰色）
│           │   ├── 任务名称（截断）
│           │   ├── 状态圆点（绿色=运行中 / 灰色=已暂停）
│           │   ├── 调度计划文字（人类可读，例："每天上午 9 点"）
│           │   └── 启用/禁用开关（Switch，独立点击区域，不触发编辑）
│           │
│           ├── 消息预览（最多显示 2 行）
│           │
│           ├── 元数据区（条件显示）
│           │   ├── 投递通道标识（delivery.mode='announce' 时）
│           │   │   ├── 通道图标 + 通道显示名称
│           │   │   └── 账号名称（有 deliveryAccountName 时）或 delivery.to 目标
│           │   ├── 上次运行（job.lastRun 存在时）
│           │   │   ├── "上次运行" + 相对时间
│           │   │   ├── 成功：绿色 ✓ 图标
│           │   │   └── 失败：红色 ✗ 图标
│           │   └── 下次运行（job.nextRun 存在 且 enabled=true 时）
│           │       └── "下次运行" + 本地时间字符串
│           │
│           ├── 上次运行错误（lastRun.success=false 且 lastRun.error 存在时）
│           │   └── 警告框：显示错误信息（最多 2 行）
│           │
│           └── 操作按钮（悬停卡片时显示）
│               ├── [立即运行]（触发中显示加载动画+禁用）
│               │   └── 成功 → Toast "任务已成功触发"
│               │   └── 失败 → Toast "触发任务失败: {{error}}"
│               └── [删除] → 打开删除确认对话框
│
├── 创建/编辑弹窗（TaskDialog）
│   ├── 头部
│   │   ├── 标题："创建任务"（新建）/ "编辑任务"（编辑）
│   │   ├── 副标题："安排自动化的 AI 任务"
│   │   └── [×] 关闭按钮（点击外部遮罩也关闭）
│   │
│   ├── 表单字段
│   │   ├── 任务名称
│   │   │   └── 输入框，占位符："例如：早间简报"
│   │   │
│   │   ├── 消息/提示词
│   │   │   └── 多行文本框（3 行），占位符："AI 应该做什么？例如：给我一份今天的新闻和天气摘要"
│   │   │
│   │   ├── 调度计划
│   │   │   ├── 预设模式（useCustom=false，默认）
│   │   │   │   ├── 8 个预设按钮（2 列网格，选中高亮）
│   │   │   │   │   ├── "每分钟"        (* * * * *)

│   │   │   │   │   ├── "每 15 分钟"    (*/15 * * * *)
│   │   │   │   │   ├── "每小时"        (0 * * * *)
│   │   │   │   │   ├── "每天上午 9 点" (0 9 * * *)
│   │   │   │   │   ├── "每天下午 6 点" (0 18 * * *)
│   │   │   │   │   ├── "每周 (周一上午 9 点)" (0 9 * * 1)
│   │   │   │   │   └── "每月 (1号上午 9 点)"  (0 9 1 * *)
│   │   │   │   └── [使用自定义 Cron] 切换链接
│   │   │   │
│   │   │   └── 自定义模式（useCustom=true）
│   │   │       ├── 输入框，占位符："Cron 表达式 (例如：0 9 * * *)"
│   │   │       └── [使用预设] 切换链接
│   │   │
│   │   │   ├── 下次运行预估（有预估时）："下次运行: {{时间}}"
│   │   │   └── 无法预估时显示占位符文字
```

## 调度计划格式说明

```
输入格式（三种）                     解析输出示例
───────────────────────────────────────────────────────
cron 字符串       "0 9 * * *"        → "每天上午 9 点"
Gateway 对象     { kind: "cron", expr: "*/5 * * * *" } → "每 5 分钟"
Gateway 对象     { kind: "every", everyMs: 3600000 }   → "每 1 小时"
Gateway 对象     { kind: "at", at: "2026-04-12T09:00:00Z" } → "执行一次，时间：{{本地时间}}"
```

```
├── "任务已暂停"
├── "任务已删除"
```

## 数据流骨架

```
页面挂载
  → isGatewayRunning=true 时：fetchJobs()（/api/cron/jobs）
  → 同时：fetchConfiguredChannels()（/api/channels/accounts）

任务操作
  → 创建：填表单 → 校验 → createJob() POST /api/cron/jobs → 更新列表
  → 编辑：点击卡片 → 打开弹窗 → 修改 → updateJob() PUT /api/cron/jobs/:id
  → 删除：确认对话框 → deleteJob() DELETE /api/cron/jobs/:id → Toast
  → 开关：Switch → toggleJob() POST /api/cron/toggle → Toast
  → 立即运行：[立即运行] → triggerJob() POST /api/cron/trigger → 刷新列表 → Toast

投递目标加载（deliveryMode='announce' 时）
  → 切换通道/账号时自动触发
  → GET /api/channels/targets?channelType=&accountId=
  → 返回 targets 列表 → 填充下拉菜单
  → 切换通道/账号时重置已选目标
```
