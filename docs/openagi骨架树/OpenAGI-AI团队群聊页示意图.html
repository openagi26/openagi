安全工程师 · Claude Haiku · 0.3
OAuth 2.0 + PKCE 标准流程，建议：

1. 微信OAuth走服务端流程（不暴露 AppSecret）
2. 手机号用短信验证码（60秒防刷 + 图形验证码）
3. 所有 token 存 HttpOnly Secure Cookie
4. 增加设备指纹识别，异常登录二次验证

⚠️ 注意：微信小程序和网页版的OAuth流程不同，需分别处理。
